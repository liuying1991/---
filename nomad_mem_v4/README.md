# NomadMem v4.0 — 自主AI记忆系统（人脑式架构）

## 核心设计

**从符号层下沉到亚符号层，实现人脑式分布式表征与存算一体**

### 三层架构

```
第一层：向量嵌入空间（亚符号）
  └── 多模态编码器 → 向量数据库 → 相似度检索

第二层：图关联层（符号）
  └── 概念节点 + 显式关系

第三层：双存储层（快写+慢固）
  └── 海马体快写 + 皮层慢固 + 睡眠巩固
```

### 人脑特性

| 人脑事实 | v4.0实现 |
|---------|---------|
| 存算一体 | 检索=重建，不是读取 |
| 情绪门控 | 情绪分数×学习率 |
| 主动遗忘 | 衰减+修剪+合并 |
| 双阶段记忆 | 海马体→皮层迁移 |
| 工作记忆瓶颈 | 硬性4项限制 |

## 快速开始

```bash
pip install -r requirements.txt
python tests/test_v4_pipeline.py
python run.py
```
