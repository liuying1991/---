# 声音与图像特征提取计算公式详解

## 一、声音特征提取计算公式

### 1. 音调/基频(Fundamental Frequency)计算

声音的高低主要由基频决定，基频是指声音波形中最基本的频率成分。

#### 自相关法计算基频：
```
R(τ) = Σ[n=0 to N-1-τ] x(n) * x(n+τ)
```
其中：
- R(τ) 是时间延迟τ的自相关函数
- x(n) 是音频信号在第n个采样点的值
- N 是分析帧的采样点数
- τ 是时间延迟

基频F0 = argmax_τ R(τ)，其中τ在合理的基频范围内

#### 平均幅度差函数(AMDF)：
```
D(τ) = (1/N) * Σ[n=0 to N-1-τ] |x(n) - x(n+τ)|
```
基频F0 = argmin_τ D(τ)，其中τ在合理的基频范围内

### 2. 响度/音量计算

响度是人对声音强度的主观感受，通常用分贝(dB)表示。

#### 分贝计算公式：
```
L = 10 * log10(P/P0)
```
其中：
- L 是声压级(分贝)
- P 是测量的声压
- P0 是参考声压(通常取20μPa，对应人耳对1000Hz纯音的听阈)

#### 基于信号幅度的音量计算：
1. 绝对值总和法：
```
Volume = Σ[i=0 to n-1] |si|
```
其中si是一个音帧中的第i个取样点，n是每个音帧包含的采样点数。

2. 分贝法：
```
Volume(dB) = 10 * log10(Σ[i=0 to n-1] si²)
```

### 3. 过零率(Zero Crossing Rate)计算

过零率是音频信号波形穿过零点的速率，反映声音的频率特性。

```
ZCR = (1/(2N)) * Σ[n=1 to N] |sgn(x[n]) - sgn(x[n-1])|
```
其中：
- N 是帧中的采样点数
- sgn(x) 是符号函数，x>0时为1，x<0时为-1，x=0时为0

### 4. MFCC(梅尔频率倒谱系数)计算

MFCC是语音识别中最常用的特征参数，模拟人耳对频率的感知特性。

#### 计算步骤：

1. 预加重：
```
y[n] = x[n] - α * x[n-1]  (通常α=0.97)
```

2. 分帧加窗：
```
s[n] = y[n] * w[n]  (w[n]为汉明窗)
```

3. 快速傅里叶变换(FFT)：
```
X(k) = Σ[n=0 to N-1] s[n] * e^(-j2πkn/N)
```

4. 梅尔滤波器组：
将线性频率转换为梅尔频率：
```
Mel(f) = 2595 * log10(1 + f/700)
```

5. 对数能量计算：
```
E(m) = log(Σ[k=k_m_start to k_m_end] |X(k)|² * H_m(k))
```
其中H_m(k)是第m个三角滤波器的传递函数

6. 离散余弦变换(DCT)：
```
c[i] = Σ[m=1 to M] E(m) * cos(πi(m-0.5)/M)
```
其中i=0,1,...,K-1，K是MFCC系数的个数

## 二、图像特征提取计算公式

### 1. 颜色特征提取

#### 颜色直方图：
将颜色空间分成若干区间，统计每个区间内的像素数量。

对于RGB图像，可创建24维颜色直方图向量(每个颜色通道8个区间)：
```
H[r,g,b] = Σ[i,j] δ(R[i,j]=r, G[i,j]=g, B[i,j]=b)
```
其中：
- R[i,j], G[i,j], B[i,j]是像素(i,j)的RGB值
- δ是克罗内克函数，条件为真时返回1，否则返回0

#### 颜色矩：
颜色矩使用颜色分布的低阶矩来表示颜色特征。

一阶矩(均值)：
```
μ_c = (1/N) Σ[i=1 to N] p_c(i)
```

二阶矩(标准差)：
```
σ_c = √[(1/N) Σ[i=1 to N] (p_c(i) - μ_c)²]
```

三阶矩(偏度)：
```
s_c = ∛[(1/N) Σ[i=1 to N] (p_c(i) - μ_c)³]
```
其中c是颜色通道(R,G,B)，p_c(i)是第i个像素的c通道值，N是像素总数。

### 2. 边缘检测

#### Sobel算子：
使用3×3卷积核计算图像梯度：

水平方向梯度：
```
Gx = [[-1, 0, 1],
      [-2, 0, 2],
      [-1, 0, 1]]
```

垂直方向梯度：
```
Gy = [[-1, -2, -1],
      [ 0,  0,  0],
      [ 1,  2,  1]]
```

梯度幅值：
```
G = √(Gx² + Gy²)
```

梯度方向：
```
θ = arctan(Gy/Gx)
```

#### Canny边缘检测：
1. 高斯滤波：
```
G(x,y) = (1/(2πσ²)) * e^(-(x²+y²)/(2σ²))
```

2. 非极大值抑制：
在梯度方向上，保留局部梯度最大的点。

3. 双阈值检测：
设置高阈值T_H和低阈值T_L：
- 梯度大于T_H的像素确定为边缘
- 梯度介于T_L和T_H之间的像素，如果与确定的边缘相连则保留

### 3. 形状特征提取

#### LBP(局部二值模式)：
```
LBP_P,R(x_c,y_c) = Σ[p=0 to P-1] s(g_p - g_c) * 2^p
```
其中：
- (x_c,y_c)是中心像素
- g_c是中心像素的灰度值
- g_p是周围P个等距圆周上第p个像素的灰度值
- R是圆的半径
- s(x)是符号函数，x≥0时为1，否则为0

#### HOG(方向梯度直方图)：
1. 计算图像梯度：
```
Gx(x,y) = I(x+1,y) - I(x-1,y)
Gy(x,y) = I(x,y+1) - I(x,y-1)
```

2. 计算梯度幅值和方向：
```
M(x,y) = √(Gx(x,y)² + Gy(x,y)²)
θ(x,y) = arctan(Gy(x,y)/Gx(x,y))
```

3. 将图像划分为细胞(cell)，统计每个细胞的梯度方向直方图

4. 将细胞组合成块(block)，对块内的直方图进行归一化

### 4. 深度估计

#### 双目视觉深度计算：
基于双目相机的视差计算深度：

```
z = (f * B) / d
```
其中：
- z 是物体深度(距离)
- f 是相机焦距
- B 是双目相机基线距离(两相机光心距离)
- d 是视差(同一物体点在左右图像中的像素位置差)

#### 视差计算：
```
d = x_l - x_r
```
其中x_l和x_r分别是物体点在左右图像中的水平坐标。

#### 立体匹配算法(SAD)：
```
SAD(x,y,d) = Σ[i=-w to w] Σ[j=-h to h] |I_L(x+i,y+j) - I_R(x+i-d,y+j)|
```
其中：
- (x,y)是左图像中的像素坐标
- d是视差
- I_L和I_R分别是左右图像
- w和h是匹配窗口的半宽和半高

最佳视差：
```
d* = argmin_d SAD(x,y,d)
```

## 三、综合应用示例

### 音频-视频同步处理系统

假设我们要构建一个能够同时处理音频和视频的系统，提取多模态特征：

1. **音频处理流程**：
   - 采样音频信号(44.1kHz)
   - 分帧(每帧20ms，帧移10ms)
   - 计算每帧的MFCC系数(13维)
   - 计算每帧的基频和能量
   - 组合成音频特征向量

2. **视频处理流程**：
   - 捕获视频帧(30fps)
   - 提取每帧的HOG特征
   - 进行人脸检测和关键点定位
   - 计算面部表情特征
   - 组合成视觉特征向量

3. **多模态融合**：
   - 时间对齐音频和视频特征
   - 使用加权融合或深度学习模型融合多模态特征
   - 输出综合特征向量用于后续分析

这些计算公式为构建具有类人类感知能力的系统提供了基础，通过这些数学模型，计算机可以从原始的音频和视频数据中提取有意义的特征，进而实现更高级的认知功能。
