# Consciousness Engine - 意识引擎

## 设计理念

**不是传统游戏引擎，而是模拟人类大脑的计算标准**

传统游戏引擎：渲染 → 物理 → 音效 → AI
意识引擎：感知 → 记忆 → 思考 → 决策 → 行动

## 架构

```
第四层：虚拟现实层 (VR Layer)
  └── 3D场景、交互界面、意识体呈现

第三层：意识法则层 (Consciousness Layer)
  ├── NomadMem v4.0 记忆大脑
  ├── 注意力系统 (工作记忆4项限制)
  ├── 睡眠系统 (定期巩固)
  └── 遗忘系统 (主动修剪)

第二层：生物法则层 (Bio Layer)
  ├── 感知系统 (视觉/听觉/触觉 → 神经信号)
  ├── 神经系统 (神经元发放、突触可塑性)
  ├── 情绪系统 (杏仁核→海马体门控)
  └── 运动系统 (决策→执行)

第一层：物理法则层 (Physics Layer)
  ├── 空间几何 (XYZ坐标、距离、方向)
  ├── 时间流 (线性时间、时间戳)
  ├── 因果律 (事件→结果)
  └── 守恒律 (信息守恒)
```

## 核心概念

### 意识体 (Consciousness Entity)
存在于3D空间中的虚拟意识，拥有：
- 记忆大脑（NomadMem v4.0）
- 感知系统（多模态编码）
- 情绪系统（情感门控）
- 决策系统（注意力+工作记忆）

### 物理法则
- **空间**：XYZ坐标系，距离计算，方向感知
- **时间**：线性时间流，时间戳，时序记忆
- **因果**：事件→结果关联，因果推理
- **守恒**：信息不丢失，只在衰减

### 生物法则
- **感知**：视觉/听觉/触觉 → 神经信号 → 向量嵌入
- **神经**：赫布学习（一起发放→连接更强）
- **情绪**：杏仁核调制，情绪门控学习率
- **运动**：决策→行动执行

## 快速开始

```bash
pip install -r requirements.txt
python tests/test_consciousness.py
python run.py
```
