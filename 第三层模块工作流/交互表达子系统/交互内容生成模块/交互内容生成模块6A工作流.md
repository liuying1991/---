# 交互内容生成模块6A工作流

## 模块概述

### 模块名称
交互内容生成模块

### 所属子系统
交互表达子系统

### 模块功能描述
交互内容生成模块负责根据交互策略和婴儿状态，生成适合的交互内容，包括语音内容、表情内容和动作内容。该模块通过分析婴儿的年龄、状态、偏好和习惯，结合交互策略，生成个性化、适应性和吸引力的交互内容，确保交互的有效性和趣味性。

### 模块职责
1. 内容生成：生成适合的交互内容
2. 内容管理：管理和维护内容库
3. 内容定制：定制个性化内容
4. 内容优化：优化内容效果和适应性
5. 内容安全：确保内容安全和适宜性
6. 内容评估：评估内容效果和质量

### 模块接口
- **输入接口**：交互策略、婴儿状态、环境数据、反馈数据
- **输出接口**：语音内容、表情内容、动作内容、内容参数
- **依赖模块**：交互策略模块、感知处理子系统、决策模块
- **被依赖模块**：语音合成模块、表情生成模块、肢体动作生成模块

## Ask阶段：需求分析

4. 支持基于反馈的内容优化
5. 支持内容安全和适宜性检查
6. 支持内容效果评估和优化

8. 支持内容版本管理和回滚

4. **实时性需求**：内容调整延迟不超过300ms
5. **资源需求**：内存占用不超过1GB，CPU占用不超过40%

3. **实时性约束**：内容生成和调整必须实时响应

## Analyze阶段：架构设计

1. **数据层**：负责婴儿数据、策略数据、内容数据和反馈数据的收集、存储和管理
2. **分析层**：负责婴儿状态分析、策略分析、反馈分析和环境分析
3. **生成层**：负责内容生成、内容定制、内容优化和内容安全检查
4. **输出层**：负责格式化输出和结果传递

### 数据模型
交互内容生成模块的数据模型包括：
1. **婴儿数据模型**：包含基本信息、状态信息、偏好信息、习惯信息等
2. **策略数据模型**：包含策略类型、策略参数、策略优先级等
3. **内容数据模型**：包含内容类型、内容参数、内容效果等
4. **反馈数据模型**：包含反馈记录、反馈分析、反馈历史等
5. **环境数据模型**：包含环境状态、环境变化、环境适应等

3. **自然语言处理**：NLTK/spaCy - 文本处理和分析
4. **数据处理库**：pandas/numpy - 数据处理和分析

class InteractionContentManager:
    def __init__(self, baby_analyzer, strategy_analyzer, feedback_analyzer,
                 environment_analyzer, content_generator, voice_generator,
                 expression_generator, action_generator, personalization_engine,
                 age_adapter, preference_adapter, habit_adapter, content_optimizer,
                 effectiveness_optimizer, engagement_optimizer, safety_checker,
                 appropriateness_checker, content_filter, content_manager,
                 version_manager, repository_manager, content_evaluator,
                 effectiveness_analyzer, quality_assessor):
        self.baby_analyzer = baby_analyzer
        self.strategy_analyzer = strategy_analyzer
        self.feedback_analyzer = feedback_analyzer
        self.environment_analyzer = environment_analyzer
        self.content_generator = content_generator
        self.voice_generator = voice_generator
        self.expression_generator = expression_generator
        self.action_generator = action_generator
        self.personalization_engine = personalization_engine
        self.age_adapter = age_adapter
        self.preference_adapter = preference_adapter
        self.habit_adapter = habit_adapter
        self.content_optimizer = content_optimizer
        self.effectiveness_optimizer = effectiveness_optimizer
        self.engagement_optimizer = engagement_optimizer
        self.safety_checker = safety_checker
        self.appropriateness_checker = appropriateness_checker
        self.content_filter = content_filter
        self.content_manager = content_manager
        self.version_manager = version_manager
        self.repository_manager = repository_manager
        self.content_evaluator = content_evaluator
        self.effectiveness_analyzer = effectiveness_analyzer
        self.quality_assessor = quality_assessor
        
        # 初始化模型
        self._load_models()
        
    async def generate_content(self, baby_id, session_id, strategy, context):
        try:
            # 分析婴儿状态
            baby_analysis = await self.baby_analyzer.analyze_baby(baby_id, context)
            
            # 分析策略要求
            strategy_analysis = await self.strategy_analyzer.analyze_strategy(strategy)
            
            # 分析反馈信息
            feedback_analysis = await self.feedback_analyzer.analyze_feedback(
                baby_id, session_id, context
            )
            
            # 分析环境状态
            environment_analysis = await self.environment_analyzer.analyze_environment(context)
            
            # 生成基础内容
            base_content = await self.content_generator.generate_content(
                baby_analysis, strategy_analysis, feedback_analysis, environment_analysis
            )
            
            # 生成语音内容
            voice_content = await self.voice_generator.generate_voice_content(
                base_content, baby_analysis, strategy_analysis
            )
            
            # 生成表情内容
            expression_content = await self.expression_generator.generate_expression_content(
                base_content, baby_analysis, strategy_analysis
            )
            
            # 生成动作内容
            action_content = await self.action_generator.generate_action_content(
                base_content, baby_analysis, strategy_analysis
            )
            
            # 整合内容
            integrated_content = await self._integrate_content(
                voice_content, expression_content, action_content
            )
            
            # 定制内容
            personalized_content = await self.personalization_engine.personalize_content(
                integrated_content, baby_analysis
            )
            
            # 年龄适配
            age_adapted_content = await self.age_adapter.adapt_content(
                personalized_content, baby_analysis
            )
            
            # 偏好适配
            preference_adapted_content = await self.preference_adapter.adapt_content(
                age_adapted_content, baby_analysis
            )
            
            # 习惯适配
            habit_adapted_content = await self.habit_adapter.adapt_content(
                preference_adapted_content, baby_analysis
            )
            
            # 优化内容
            optimized_content = await self.content_optimizer.optimize_content(
                habit_adapted_content, feedback_analysis, environment_analysis
            )
            
            # 安全检查
            safety_check_result = await self.safety_checker.check_safety(optimized_content)
            
            # 适宜性检查
            appropriateness_check_result = await self.appropriateness_checker.check_appropriateness(
                optimized_content, baby_analysis
            )
            
            # 内容过滤
            filtered_content = await self.content_filter.filter_content(
                optimized_content, safety_check_result, appropriateness_check_result
            )
            
            # 构建内容结果
            content_result = {
                "baby_id": baby_id,
                "session_id": session_id,
                "content": filtered_content,
                "voice_content": voice_content,
                "expression_content": expression_content,
                "action_content": action_content,
                "baby_analysis": baby_analysis,
                "strategy_analysis": strategy_analysis,
                "feedback_analysis": feedback_analysis,
                "environment_analysis": environment_analysis,
                "safety_check": safety_check_result,
                "appropriateness_check": appropriateness_check_result,
                "timestamp": time.time()
            }
            
            return content_result
            
        except Exception as e:
            # 处理异常
            error_result = {
                "baby_id": baby_id,
                "session_id": session_id,
                "error": str(e),
                "timestamp": time.time()
            }
            
            return error_result
    
    async def adjust_content(self, baby_id, session_id, current_content, feedback_data):
        try:
            # 分析反馈数据
            feedback_analysis = await self.feedback_analyzer.analyze_feedback_data(
                baby_id, session_id, feedback_data
            )
            
            # 优化内容效果
            effectiveness_optimized_content = await self.effectiveness_optimizer.optimize_effectiveness(
                current_content, feedback_analysis
            )
            
            # 优化内容参与度
            engagement_optimized_content = await self.engagement_optimizer.optimize_engagement(
                effectiveness_optimized_content, feedback_analysis
            )
            
            # 安全检查
            safety_check_result = await self.safety_checker.check_safety(
                engagement_optimized_content
            )
            
            # 适宜性检查
            appropriateness_check_result = await self.appropriateness_checker.check_appropriateness(
                engagement_optimized_content, await self.baby_analyzer.analyze_baby(baby_id, {})
            )
            
            # 内容过滤
            filtered_content = await self.content_filter.filter_content(
                engagement_optimized_content, safety_check_result, appropriateness_check_result
            )
            
            # 构建调整结果
            adjustment_result = {
                "baby_id": baby_id,
                "session_id": session_id,
                "original_content": current_content,
                "adjusted_content": filtered_content,
                "feedback_analysis": feedback_analysis,
                "safety_check": safety_check_result,
                "appropriateness_check": appropriateness_check_result,
                "timestamp": time.time()
            }
            
            return adjustment_result
            
        except Exception as e:
            # 处理异常
            error_result = {
                "baby_id": baby_id,
                "session_id": session_id,
                "error": str(e),
                "timestamp": time.time()
            }
            
            return error_result

class SafetyChecker:
    def __init__(self, safety_model, safety_rules):
        self.safety_model = safety_model
        self.safety_rules = safety_rules
        
    async def check_safety(self, content):
        try:
            # 提取内容特征
            content_features = await self._extract_content_features(content)
            
            # 模型预测安全性
            model_prediction = await self._predict_safety(content_features)
            
            # 应用安全规则
            rule_results = await self._apply_safety_rules(content)
            
            # 合并检查结果
            safety_result = await self._merge_safety_results(
                model_prediction, rule_results
            )
            
            return {
                "content": content,
                "content_features": content_features,
                "model_prediction": model_prediction,
                "rule_results": rule_results,
                "safety_result": safety_result,
                "is_safe": safety_result.get("is_safe", False),
                "safety_score": safety_result.get("safety_score", 0.0),
                "safety_issues": safety_result.get("safety_issues", [])
            }
            
        except Exception as e:
            return {
                "error": str(e),
                "is_safe": False,
                "safety_score": 0.0,
                "safety_issues": ["Safety check failed"]
            }
    
    async def _extract_content_features(self, content):
        # 提取语音内容特征
        voice_content = content.get("voice", {})

        # 提取表情内容特征

        # 合并特征
        content_features = {
            **voice_features,
            **expression_features,
            **action_features
        }
        
        return content_features
    
    async def _extract_voice_features(self, voice_content):
        # 提取语音文本特征
        text = voice_content.get("text", "")
        text_features = await self._extract_text_features(text)
        
        # 提取语音参数特征

            "text_words": len(text.split()),

            "speed": speed,

    async def _extract_text_features(self, text):
        # 提取文本特征
        # 这里简化处理，实际应该使用NLP技术提取更丰富的特征
        
        # 检查敏感词
        sensitive_words = ["violence", "scary", "inappropriate"]
        sensitive_count = sum(1 for word in sensitive_words if word in text.lower())
        
        # 检查复杂度
        complex_words = [word for word in text.split() if len(word) > 6]
        complex_count = len(complex_words)
        
        return {
            "sensitive_word_count": sensitive_count,

            "has_sensitive_words": sensitive_count > 0,

        }
    
    async def _predict_safety(self, content_features):
        # 转换特征为模型输入格式
        feature_vector = self._convert_features_to_vector(content_features)
        
        # 模型预测
        safety_score = self.safety_model.predict_proba([feature_vector])[0][1]
        
        return {
            "safety_score": safety_score,
            "is_safe": safety_score > 0.7
        }
    
    async def _apply_safety_rules(self, content):
        rule_results = []
        
        # 规则1：检查语音音量
        voice_content = content.get("voice", {})
        volume = voice_content.get("volume", 0.8)
        if volume > 0.9:
            rule_results.append({
                "rule": "volume_check",
                "passed": False,
                "reason": "Voice volume too high",
                "suggestion": "Reduce volume to 0.8 or lower"
            })
        
        # 规则2：检查表情强度
        expression_content = content.get("expression", {})
        intensity = expression_content.get("intensity", 0.7)
        if intensity > 0.9:
            rule_results.append({
                "rule": "intensity_check",
                "passed": False,
                "reason": "Expression intensity too high",
                "suggestion": "Reduce intensity to 0.8 or lower"
            })
        
        # 规则3：检查动作幅度
        action_content = content.get("action", {})
        amplitude = action_content.get("amplitude", 0.7)
        if amplitude > 0.9:
            rule_results.append({
                "rule": "amplitude_check",
                "passed": False,
                "reason": "Action amplitude too high",
                "suggestion": "Reduce amplitude to 0.8 or lower"
            })
        
        # 规则4：检查文本敏感词
        text = voice_content.get("text", "")
        sensitive_words = ["violence", "scary", "inappropriate"]
        found_sensitive = [word for word in sensitive_words if word in text.lower()]
        if found_sensitive:
            rule_results.append({
                "rule": "sensitive_words_check",
                "passed": False,
                "reason": f"Sensitive words found: {', '.join(found_sensitive)}",
                "suggestion": "Remove or replace sensitive words"
            })
        
        return rule_results
    
    async def _merge_safety_results(self, model_prediction, rule_results):
        # 获取模型预测结果
        model_safety_score = model_prediction.get("safety_score", 0.0)
        model_is_safe = model_prediction.get("is_safe", False)
        
        # 检查规则结果
        failed_rules = [rule for rule in rule_results if not rule.get("passed", True)]
        
        # 计算最终安全分数
        if failed_rules:
            # 有规则失败，降低安全分数
            final_safety_score = max(model_safety_score - 0.3, 0.0)
            final_is_safe = False
        else:
            # 所有规则通过，使用模型预测结果
            final_safety_score = model_safety_score
            final_is_safe = model_is_safe
        
        # 收集安全问题
        safety_issues = []
        
        if not model_is_safe:
            safety_issues.append("Model prediction indicates unsafe content")
        
        for rule in failed_rules:
            safety_issues.append(rule.get("reason", "Unknown safety issue"))
        
        return {
            "safety_score": final_safety_score,
            "is_safe": final_is_safe,
            "safety_issues": safety_issues,
            "failed_rules": failed_rules
        }

        feature_names = sorted(features.keys())
        feature_vector = [features[name] for name in feature_names]
        
        return feature_vector
