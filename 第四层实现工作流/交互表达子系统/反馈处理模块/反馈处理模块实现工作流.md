## 错误处理

所有API错误响应都遵循以下格式：

```json
{
  "success": false,
  "error": {
    "code": "ERROR_CODE",
    "message": "错误描述",
    "details": "详细错误信息"
  }
}
```

常见错误代码：

- `INVALID_REQUEST`: 请求参数无效
- `UNAUTHORIZED`: 认证失败
- `FORBIDDEN`: 权限不足
- `NOT_FOUND`: 资源不存在
- `INTERNAL_ERROR`: 服务器内部错误
- `SERVICE_UNAVAILABLE`: 服务不可用

## 限流

API实施了请求限流策略：

- 每个API密钥每分钟最多100个请求
- 超过限制将返回`429 Too Many Requests`状态码
- 限流重置时间包含在响应头`X-RateLimit-Reset`中

## 性能优化

### 1. 异步处理

使用异步处理提高并发性能：

```python
import asyncio

async def process_multiple_feedbacks(self, feedback_list):
    """并行处理多个反馈"""
    tasks = [
        self.process_feedback(
            fb["session_id"], 
            fb["interaction_id"], 
            fb["feedback_data"]
        )
        for fb in feedback_list
    ]
    
    results = await asyncio.gather(*tasks, return_exceptions=True)
    return results
```

### 2. 缓存策略

使用Redis缓存频繁访问的数据：

```python
import aioredis

async def get_cached_analysis(self, feedback_id: str):
    """获取缓存的分析结果"""
    redis = await aioredis.create_redis_pool('redis://localhost')
    
    try:
        cached_result = await redis.get(f"feedback_analysis:{feedback_id}")
        if cached_result:
            return json.loads(cached_result)
    finally:
        redis.close()
        await redis.wait_closed()
        
    return None

async def cache_analysis(self, feedback_id: str, analysis_result):
    """缓存分析结果"""
    redis = await aioredis.create_redis_pool('redis://localhost')
    
    try:
        await redis.setex(
            f"feedback_analysis:{feedback_id}", 
            3600,  # 1小时过期
            json.dumps(analysis_result)
        )
    finally:
        redis.close()
        await redis.wait_closed()
```

### 3. 批处理

使用批处理提高处理效率：

```python
async def batch_process_feedbacks(self, feedback_batch):
    """批量处理反馈"""
    # 预处理阶段
    preprocessed_batch = await self._batch_preprocess(feedback_batch)
    
    # 特征提取阶段
    features_batch = await self._batch_extract_features(preprocessed_batch)
    
    # 分析阶段
    analysis_batch = await self._batch_analyze(features_batch)
    
    return analysis_batch
```

# src/core/feedback_collector.py
import asyncio
import time
from typing import Dict, List, Optional, Any
from dataclasses import dataclass
import numpy as np

@dataclass
class FeedbackData:
    """反馈数据结构"""
    session_id: str
    interaction_id: str
    timestamp: float
    facial_feedback: Optional[Dict[str, Any]] = None
    vocal_feedback: Optional[Dict[str, Any]] = None
    gesture_feedback: Optional[Dict[str, Any]] = None
    environmental_feedback: Optional[Dict[str, Any]] = None

class FeedbackCollector:
    """反馈收集器 - 负责收集多模态反馈数据"""
    
    def __init__(self, config: Dict[str, Any]):
        self.config = config
        self.active_sessions = {}
        self.collection_queue = asyncio.Queue()
        self.is_running = False
        
    async def start_collection(self):
        """启动反馈收集服务"""
        self.is_running = True
        # 启动收集任务
        asyncio.create_task(self._collection_worker())
        
    async def stop_collection(self):
        """停止反馈收集服务"""
        self.is_running = False
        
    async def collect_feedback(self, session_id: str, interaction_id: str, 
                             raw_feedback: Dict[str, Any]) -> FeedbackData:
        """收集反馈数据"""
        try:
            # 创建反馈数据对象
            feedback_data = FeedbackData(
                session_id=session_id,
                interaction_id=interaction_id,
                timestamp=time.time()
            )
            
            # 处理面部反馈
            if 'facial' in raw_feedback:
                feedback_data.facial_feedback = await self._process_facial_feedback(
                    raw_feedback['facial']
                )
                
            # 处理语音反馈
            if 'vocal' in raw_feedback:
                feedback_data.vocal_feedback = await self._process_vocal_feedback(
                    raw_feedback['vocal']
                )
                
            # 处理手势反馈
            if 'gesture' in raw_feedback:
                feedback_data.gesture_feedback = await self._process_gesture_feedback(
                    raw_feedback['gesture']
                )
                
            # 处理环境反馈
            if 'environmental' in raw_feedback:
                feedback_data.environmental_feedback = await self._process_environmental_feedback(
                    raw_feedback['environmental']
                )
                
            return feedback_data
            
        except Exception as e:
            raise Exception(f"反馈收集失败: {str(e)}")
            
    async def _process_facial_feedback(self, facial_data: Dict[str, Any]) -> Dict[str, Any]:
        """处理面部反馈数据"""
        # 提取面部表情特征
        # 这里可以调用面部表情识别API或模型
        return {
            "expression": facial_data.get("expression", "neutral"),
            "intensity": facial_data.get("intensity", 0.5),
            "confidence": facial_data.get("confidence", 0.8),
            "keypoints": facial_data.get("keypoints", []),
            "timestamp": time.time()
        }
        
    async def _process_vocal_feedback(self, vocal_data: Dict[str, Any]) -> Dict[str, Any]:
        """处理语音反馈数据"""
        # 提取语音特征

            if len(keypoints) > 0:
                # 计算面部中心

                normalized_keypoints = np.zeros_like(keypoints)
                normalized_keypoints[:, 0] = (keypoints[:, 0] - center_x) / face_size

                intensity = facial_feedback.get("intensity", 0.5)
                confidence = facial_feedback.get("confidence", 0.8)
                
                # 构建特征向量
                features = np.concatenate([
                    normalized_keypoints.flatten(),
                    [intensity, confidence]
                ])
                
                return features
            else:
                # 如果没有关键点，返回默认特征
                return np.zeros([68 * 2 + 2])  # 68个关键点 + 强度 + 置信度

                combined = combined.unsqueeze(0)  # 添加序列维度
                attended, _ = self.attention(combined, combined, combined)
                attended = attended.squeeze(0)  # 移除序列维度
                
                # 情感分类
                emotion = self.relu(self.emotion_fc1(attended))
                emotion = self.dropout(emotion)
                emotion = self.emotion_fc2(emotion)
                
                return emotion
        
        return MultimodalEmotionModel(64, 32, 16, 16, len(self.emotion_labels))
        
    async def analyze_emotion(self, features) -> EmotionAnalysis:
        """分析情感"""
        try:
            # 创建情感分析结果对象
            analysis = EmotionAnalysis(
                session_id=features.session_id,
                interaction_id=features.interaction_id,
                timestamp=features.timestamp
            )
            
            # 初始化情感分数
            emotion_scores = {emotion: 0.0 for emotion in self.emotion_labels}
            
            # 面部情感分析
            facial_emotion_scores = None
            if features.facial_features is not None:
                facial_tensor = torch.FloatTensor(features.facial_features).unsqueeze(0)
                with torch.no_grad():
                    facial_logits = self.facial_emotion_model(facial_tensor)
                    facial_probs = F.softmax(facial_logits, dim=-1)
                    facial_emotion_scores = facial_probs.squeeze(0).numpy()
                    
                    # 更新情感分数
                    for i, emotion in enumerate(self.emotion_labels):
                        emotion_scores[emotion] += facial_emotion_scores[i] * 0.4  # 面部权重
                        
            # 语音情感分析
            vocal_emotion_scores = None
            if features.vocal_features is not None:
                vocal_tensor = torch.FloatTensor(features.vocal_features).unsqueeze(0)
                with torch.no_grad():
                    vocal_logits = self.vocal_emotion_model(vocal_tensor)
                    vocal_probs = F.softmax(vocal_logits, dim=-1)
                    vocal_emotion_scores = vocal_probs.squeeze(0).numpy()
                    
                    # 更新情感分数
                    for i, emotion in enumerate(self.emotion_labels):
                        emotion_scores[emotion] += vocal_emotion_scores[i] * 0.3  # 语音权重
                        
            # 多模态情感分析
            multimodal_emotion_scores = None
            if (features.facial_features is not None and 
                features.vocal_features is not None and 
                features.gesture_features is not None and 
                features.environmental_features is not None):
                
                facial_tensor = torch.FloatTensor(features.facial_features).unsqueeze(0)
                vocal_tensor = torch.FloatTensor(features.vocal_features).unsqueeze(0)
                gesture_tensor = torch.FloatTensor(features.gesture_features).unsqueeze(0)
                env_tensor = torch.FloatTensor(features.environmental_features).unsqueeze(0)
                
                with torch.no_grad():
                    multimodal_logits = self.multimodal_emotion_model(
                        facial_tensor, vocal_tensor, gesture_tensor, env_tensor
                    )
                    multimodal_probs = F.softmax(multimodal_logits, dim=-1)
                    multimodal_emotion_scores = multimodal_probs.squeeze(0).numpy()
                    
                    # 更新情感分数
                    for i, emotion in enumerate(self.emotion_labels):
                        emotion_scores[emotion] += multimodal_emotion_scores[i] * 0.3  # 多模态权重
                        
            # 确定主要情感
            primary_emotion = max(emotion_scores, key=emotion_scores.get)
            confidence = emotion_scores[primary_emotion]
            
            # 确定多模态主要情感
            if multimodal_emotion_scores is not None:
                multimodal_emotion_idx = np.argmax(multimodal_emotion_scores)
                multimodal_emotion = self.emotion_labels[multimodal_emotion_idx]
                multimodal_confidence = multimodal_emotion_scores[multimodal_emotion_idx]
            else:
                multimodal_emotion = primary_emotion
                multimodal_confidence = confidence
                
            # 更新分析结果
            analysis.primary_emotion = primary_emotion
            analysis.emotion_scores = emotion_scores
            analysis.confidence = confidence
            analysis.multimodal_emotion = multimodal_emotion
            analysis.multimodal_confidence = multimodal_confidence
            
            return analysis
            
        except Exception as e:
            raise Exception(f"情感分析失败: {str(e)}")

# src/core/intent_recognizer.py
import asyncio
import numpy as np
from typing import Dict, List, Optional, Any, Tuple
from dataclasses import dataclass
import torch
import torch.nn as nn
import torch.nn.functional as F

@dataclass
class IntentRecognition:
    """意图识别结果"""
    session_id: str
    interaction_id: str
    primary_intent: str
    intent_scores: Dict[str, float]
    confidence: float
    context_intents: List[str]
    timestamp: float = 0.0

class IntentRecognizer:
    """意图识别器 - 负责识别反馈中的意图信息"""
    
    def __init__(self, config: Dict[str, Any]):
        self.config = config
        self.intent_labels = [
            "attention", "comfort", "play", "feed", "sleep", 
            "change", "explore", "communicate", "none"
        ]
        self.context_window = config.get("context_window", 5)
        self.intent_model = self._init_intent_model()
        self.context_model = self._init_context_model()
        
    def _init_intent_model(self):
        """初始化意图识别模型"""
        # 初始化意图识别模型
        class IntentModel(nn.Module):
            def __init__(self, input_size, hidden_size, num_intents):
                super(IntentModel, self).__init__()
                self.lstm = nn.LSTM(input_size, hidden_size, batch_first=True)
                self.fc = nn.Linear(hidden_size, num_intents)
                
            def forward(self, x):
                # x shape: (batch_size, sequence_length, input_size)
                lstm_out, (h_n, c_n) = self.lstm(x)
                # 使用最后一个时间步的输出
                last_output = lstm_out[:, -1, :]
                intent_logits = self.fc(last_output)
                return intent_logits
        
        return IntentModel(128, 128, len(self.intent_labels))
        
    def _init_context_model(self):
        """初始化上下文模型"""
        # 初始化上下文模型
        class ContextModel(nn.Module):
            def __init__(self, intent_size, context_size, num_intents):
                super(ContextModel, self).__init__()
                self.context_fc = nn.Linear(context_size, 64)
                self.intent_fc = nn.Linear(intent_size, 64)
                self.fusion_fc = nn.Linear(128, num_intents)
                self.relu = nn.ReLU()
                self.dropout = nn.Dropout(0.2)
                
            def forward(self, intent_features, context_features):
                intent_out = self.relu(self.intent_fc(intent_features))
                context_out = self.relu(self.context_fc(context_features))
                combined = torch.cat([intent_out, context_out], dim=-1)
                fused = self.relu(combined)
                fused = self.dropout(fused)
                context_logits = self.fusion_fc(fused)
                return context_logits
        
        return ContextModel(len(self.intent_labels), 64, len(self.intent_labels))
        
    async def recognize_intent(self, features, emotion_analysis, context_history=None) -> IntentRecognition:
        """识别意图"""
        try:
            # 创建意图识别结果对象
            recognition = IntentRecognition(
                session_id=features.session_id,
                interaction_id=features.interaction_id,
                timestamp=features.timestamp
            )
            
            # 初始化意图分数
            intent_scores = {intent: 0.0 for intent in self.intent_labels}
            
            # 使用多模态特征进行意图识别
            if features.multimodal_features is not None:
                # 准备输入数据 (假设为单个时间步)
                input_tensor = torch.FloatTensor(features.multimodal_features).unsqueeze(0).unsqueeze(0)
                
                with torch.no_grad():
                    intent_logits = self.intent_model(input_tensor)
                    intent_probs = F.softmax(intent_logits, dim=-1)
                    intent_probs_array = intent_probs.squeeze(0).numpy()
                    
                    # 更新意图分数
                    for i, intent in enumerate(self.intent_labels):
                        intent_scores[intent] = intent_probs_array[i]
                        
            # 考虑情感信息调整意图分数
            if emotion_analysis:
                intent_scores = self._adjust_intent_by_emotion(
                    intent_scores, emotion_analysis.emotion_scores
                )
                
            # 考虑上下文信息调整意图分数
            context_intents = []
            if context_history and len(context_history) > 0:
                context_intents = self._analyze_context(context_history)
                intent_scores = self._adjust_intent_by_context(
                    intent_scores, context_intents
                )
                
            # 确定主要意图
            primary_intent = max(intent_scores, key=intent_scores.get)
            confidence = intent_scores[primary_intent]
            
            # 更新识别结果
            recognition.primary_intent = primary_intent
            recognition.intent_scores = intent_scores
            recognition.confidence = confidence
            recognition.context_intents = context_intents
            
            return recognition
            
        except Exception as e:
            raise Exception(f"意图识别失败: {str(e)}")
            
    def _adjust_intent_by_emotion(self, intent_scores, emotion_scores):
        """根据情感信息调整意图分数"""
        # 定义情感与意图的关联权重
        emotion_intent_weights = {
            "happy": {"play": 0.3, "communicate": 0.2},
            "sad": {"comfort": 0.4, "attention": 0.2},
            "angry": {"comfort": 0.3, "change": 0.2},
            "surprised": {"explore": 0.3, "play": 0.2},
            "fearful": {"comfort": 0.4, "attention": 0.3},
            "neutral": {"play": 0.1, "explore": 0.1},
            "disgusted": {"change": 0.3, "feed": 0.2}
        }
        
        # 根据情感调整意图分数
        for emotion, emotion_score in emotion_scores.items():
            if emotion in emotion_intent_weights:
                for intent, weight in emotion_intent_weights[emotion].items():
                    if intent in intent_scores:
                        intent_scores[intent] += emotion_score * weight
                        
        # 归一化分数
        total_score = sum(intent_scores.values())
        if total_score > 0:
            for intent in intent_scores:
                intent_scores[intent] /= total_score
                
        return intent_scores
        
    def _analyze_context(self, context_history):
        """分析上下文历史"""
        # 提取上下文中的意图
        context_intents = []
        for item in context_history:
            if "intent" in item:
                context_intents.append(item["intent"])
                
        return context_intents
        
    def _adjust_intent_by_context(self, intent_scores, context_intents):
        """根据上下文信息调整意图分数"""
        # 统计上下文中各意图的出现频率
        intent_counts = {}
        for intent in context_intents:
            if intent in intent_counts:
                intent_counts[intent] += 1
            else:
                intent_counts[intent] = 1
                
        # 根据上下文频率调整意图分数
        if len(context_intents) > 0:
            for intent, count in intent_counts.items():
                frequency = count / len(context_intents)
                if intent in intent_scores:
                    # 上下文权重随时间衰减
                    context_weight = 0.2 * frequency
                    intent_scores[intent] += context_weight
                    
        # 归一化分数
        total_score = sum(intent_scores.values())
        if total_score > 0:
            for intent in intent_scores:
                intent_scores[intent] /= total_score
                
        return intent_scores

# 反馈处理模块API文档

## 概述

反馈处理模块API提供了一系列接口，用于收集、分析和处理婴儿对交互表达的各种反馈，包括表情、语音、动作等多模态反馈信息，以及环境反馈信息。

## 基础信息

- **Base URL**: `https://api.ai-babysitter.com/feedback-processing/v1`
- **认证方式**: Bearer Token (JWT)
- **数据格式**: JSON
- **字符编码**: UTF-8

## 认证

所有API请求都需要在Header中包含有效的JWT Token：

```
Authorization: Bearer <your_jwt_token>
```

## API端点

### 1. 处理反馈

处理婴儿的多模态反馈数据。

**请求**
- **URL**: `/api/v1/feedback/process`
- **方法**: `POST`
- **认证**: 必需

**请求体**
```json
{
  "session_id": "session_001",
  "interaction_id": "interaction_001",
  "feedback_data": {
    "facial": {
      "expression": "happy",

      "keypoints": [[100, 100], [110, 100], [105, 110]]
    },

    },

      "type": "wave",
      "intensity": 0.6,
      "direction": "right"
    },
    "environmental": {
      "noise_level": 0.3,
      "lighting": "bright",

      "distractions": ["toy", "sound"]
    }
  },
  "context_history": [
    {
      "interaction_id": "interaction_000",
      "intent": "play",
      "timestamp": 1672531200.0
    }
  ]
}

**响应**
```json
{
  "success": true,

**状态码**
- `200`: 成功
- `400`: 请求参数错误
- `401`: 认证失败
- `500`: 服务器内部错误
