# 开源音视频处理项目资源汇总

## 1. 计算机视觉开源项目

### 1.1 OpenCV
- **官方网站**: https://opencv.org/
- **GitHub仓库**: https://github.com/opencv/opencv
- **文档**: https://docs.opencv.org/
- **特点**: 
  - 跨平台计算机视觉库
  - 支持C++、Python、Java等多种编程语言
  - 提供丰富的图像处理和计算机视觉算法
  - 包含摄像头接口，可直接访问摄像头设备

### 1.2 V4L2 (Video for Linux Two)
- **官方网站**: https://www.linuxtv.org/wiki/index.php/V4l2
- **Linux内核文档**: https://www.kernel.org/doc/html/latest/userspace-api/media/v4l/v4l2.html
- **特点**:
  - Linux内核自带的视频设备驱动框架
  - 提供统一的API访问视频设备
  - 支持USB摄像头、网络摄像头等多种设备
  - 提供底层硬件访问能力

### 1.3 SimpleCV
- **官方网站**: http://simplecv.org/
- **GitHub仓库**: https://github.com/sightmachine/SimpleCV
- **特点**:
  - 基于OpenCV的简化计算机视觉库
  - 提供更友好的Python API
  - 适合快速原型开发

## 2. 音视频处理开源项目

### 2.1 FFmpeg
- **官方网站**: https://ffmpeg.org/
- **GitHub仓库**: https://github.com/FFmpeg/FFmpeg
- **文档**: https://ffmpeg.org/documentation.html
- **特点**:
  - 完整的音视频处理框架
  - 支持几乎所有音视频格式的编解码
  - 提供命令行工具和开发库
  - 支持音视频转码、剪辑、滤镜等操作

### 2.2 GStreamer
- **官方网站**: https://gstreamer.freedesktop.org/
- **GitHub仓库**: https://github.com/GStreamer/gstreamer
- **文档**: https://gstreamer.freedesktop.org/documentation/
- **特点**:
  - 多媒体框架，支持流媒体处理
  - 基于插件和管道的架构
  - 支持实时音视频处理
  - 跨平台支持

### 2.3 VLC Media Player
- **官方网站**: https://www.videolan.org/vlc/
- **GitHub仓库**: https://github.com/videolan/vlc
- **特点**:
  - 跨平台多媒体播放器
  - 基于libVLC库，可用于开发
  - 支持几乎所有音视频格式
  - 内置编解码器库

### 2.4 WebRTC
- **官方网站**: https://webrtc.org/
- **GitHub仓库**: https://github.com/webrtc-hacks/adapter
- **特点**:
  - 实时通信技术
  - 支持浏览器端音视频采集和处理
  - 提供JavaScript API
  - 支持P2P音视频通信

## 总结

以上列出的开源项目涵盖了从底层硬件访问到高级AI应用的完整音视频处理技术栈。根据具体需求，可以选择合适的项目进行组合使用：

1. **底层硬件访问**：V4L2、ALSA、libusb
2. **音视频处理**：FFmpeg、GStreamer、OpenCV
3. **实时通信**：WebRTC、Janus、Kurento
4. **AI应用**：TensorFlow、PyTorch、MediaPipe
5. **开发框架**：Qt、SDL、PyAV

这些开源项目提供了丰富的功能和灵活的接口，可以满足各种音视频处理应用的需求。通过合理选择和组合这些项目，可以构建出功能强大、性能优异的音视频处理系统。
