# 短期记忆模块6A工作流

## 模块概述

### 模块名称
短期记忆模块

### 所属子系统
记忆存储子系统

### 模块功能描述
短期记忆模块是记忆存储子系统的核心组件之一，负责临时存储和管理系统近期获取的信息。该模块提供高速访问能力，支持快速读写操作，并实现短期记忆到中长期记忆的自动转换机制。短期记忆模块主要处理实时感知数据、临时计算结果和近期交互信息，为系统提供即时的上下文支持。

### 模块职责
1. 临时存储系统近期获取的多模态数据
2. 提供高速访问和检索短期记忆的能力
3. 实现短期记忆的生命周期管理
4. 管理短期记忆到中长期记忆的转换
5. 优化短期记忆存储空间利用
6. 提供短期记忆的优先级管理机制

### 模块接口
- **输入接口**：接收来自感知处理子系统的多模态数据、来自认知决策子系统的临时计算结果
- **输出接口**：向中期记忆模块提供转换后的记忆数据、向认知决策子系统提供上下文信息
- **依赖模块**：感知处理子系统、认知决策子系统、中期记忆模块
- **被依赖模块**：认知决策子系统、交互表达子系统

## Analyze阶段：架构设计

### 系统架构
短期记忆模块采用分层架构设计，主要包括以下层次：

1. **接口层**：提供对外API接口，处理外部请求
2. **服务层**：实现核心业务逻辑，包括记忆存储、检索、转换等
3. **数据访问层**：封装数据存储操作，提供统一的数据访问接口
4. **存储层**：基于Redis的内存存储引擎，支持多种数据结构

模块内部采用微服务架构，各组件通过消息队列进行通信，实现松耦合设计。

### 数据模型
短期记忆模块主要包含以下数据模型：

1. **记忆项模型**：
   - 记忆ID：唯一标识符
   - 记忆类型：文本、图像、音频、视频等
   - 记忆内容：实际数据内容
   - 元数据：创建时间、访问次数、优先级等
   - 关联信息：与其他记忆的关联关系

2. **记忆索引模型**：
   - 索引ID：唯一标识符
   - 索引类型：关键词、特征向量等
   - 索引内容：具体的索引值
   - 记忆引用：指向记忆项的引用

3. **记忆转换模型**：
   - 转换ID：唯一标识符
   - 源记忆ID：待转换的记忆ID
   - 目标记忆ID：转换后的记忆ID
   - 转换条件：触发转换的条件
   - 转换状态：转换的当前状态

### 模块设计
1. **核心组件**：
   - 记忆存储组件：负责记忆数据的存储和管理
   - 记忆检索组件：负责记忆数据的检索和查询
   - 记忆转换组件：负责短期记忆到中长期记忆的转换
   - 记忆清理组件：负责过期记忆的清理和空间回收
   - 记忆索引组件：负责记忆数据的索引建立和维护

   - 各组件通过消息队列进行异步通信

   - 记忆监控组件独立运行，收集各组件的性能指标

3. **数据流**：
   - 外部数据通过接口层进入服务层
   - 服务层对数据进行处理后，通过数据访问层存储到存储层

4. **控制流**：
   - API请求由接口层接收并路由到相应的服务
   - 服务层执行业务逻辑，调用数据访问层进行数据操作
   - 记忆转换组件定期检查记忆状态，触发转换流程
   - 记忆清理组件定期清理过期记忆，释放存储空间

```python
# 记忆服务核心实现
class MemoryService:
    def __init__(self, redis_client: RedisClient, conversion_service: ConversionService):
        self.redis_client = redis_client
        self.conversion_service = conversion_service
        self.logger = get_logger(__name__)
    
    async def store_memory(self, memory_data: MemoryCreate) -> Memory:
        """存储记忆数据"""
        # 生成记忆ID
        memory_id = generate_uuid()
        
        # 创建记忆对象
        memory = Memory(
            id=memory_id,
            type=memory_data.type,
            content=memory_data.content,
            metadata=MemoryMetadata(
                created_at=datetime.utcnow(),
                access_count=0,
                priority=memory_data.priority,
                tags=memory_data.tags
            )
        )
        
        # 存储到Redis
        await self.redis_client.store_memory(memory)
        
        # 创建索引
        await self._create_indexes(memory)
        
        # 记录日志
        self.logger.info(f"Stored memory {memory_id} of type {memory.type}")
        
        return memory
    
    async def get_memory(self, memory_id: str) -> Optional[Memory]:
        """获取记忆数据"""
        memory = await self.redis_client.get_memory(memory_id)
        
        if memory:
            # 更新访问计数
            memory.metadata.access_count += 1
            memory.metadata.last_accessed = datetime.utcnow()
            await self.redis_client.update_memory_metadata(memory_id, memory.metadata)
        
        return memory
    
    async def search_memories(self, query: SearchQuery) -> List[Memory]:
        """搜索记忆数据"""
        # 根据查询类型选择搜索策略
        if query.type == SearchType.KEYWORD:
            return await self._keyword_search(query)
        elif query.type == SearchType.SIMILARITY:
            return await self._similarity_search(query)
        elif query.type == SearchType.HYBRID:
            return await self._hybrid_search(query)
        else:
            raise ValueError(f"Unsupported search type: {query.type}")
    
    async def _keyword_search(self, query: SearchQuery) -> List[Memory]:
        """基于关键词的搜索"""
        # 使用Redis的全文搜索功能
        memory_ids = await self.redis_client.search_by_keywords(query.keywords)
        
        # 获取记忆数据
        memories = []
        for memory_id in memory_ids:
            memory = await self.redis_client.get_memory(memory_id)
            if memory:
                memories.append(memory)
        
        # 按相关性排序
        memories.sort(key=lambda m: self._calculate_relevance(m, query.keywords), reverse=True)
        
        # 应用分页
        start = (query.page - 1) * query.page_size
        end = start + query.page_size
        return memories[start:end]
    
    async def _similarity_search(self, query: SearchQuery) -> List[Memory]:
        """基于相似度的搜索"""
        # 计算查询向量
        query_vector = await self._calculate_vector(query.content)
        
        # 在向量索引中搜索
        similar_memory_ids = await self.redis_client.search_by_vector(query_vector, query.threshold)
        
        # 获取记忆数据
        memories = []
        for memory_id, similarity in similar_memory_ids:
            memory = await self.redis_client.get_memory(memory_id)
            if memory:
                memory.similarity = similarity
                memories.append(memory)
        
        # 按相似度排序
        memories.sort(key=lambda m: m.similarity, reverse=True)
        
        # 应用分页
        start = (query.page - 1) * query.page_size
        end = start + query.page_size
        return memories[start:end]
    
    async def _create_indexes(self, memory: Memory):
        """为记忆创建索引"""
        # 创建关键词索引
        if memory.type == MemoryType.TEXT:
            keywords = await self._extract_keywords(memory.content)
            for keyword in keywords:
                await self.redis_client.add_keyword_index(keyword, memory.id)
        
        # 创建向量索引
        vector = await self._calculate_vector(memory.content)
        await self.redis_client.add_vector_index(memory.id, vector)
        
        # 创建时间索引
        await self.redis_client.add_time_index(memory.metadata.created_at, memory.id)
        
        # 创建标签索引
        for tag in memory.metadata.tags:
            await self.redis_client.add_tag_index(tag, memory.id)
```

3. **部署脚本**：
   ```bash
   #!/bin/bash
   # deploy.sh
   
   # 设置变量
   IMAGE_NAME="short-term-memory"
   IMAGE_TAG=${1:-latest}
   NAMESPACE="memory-system"
   
   # 构建镜像
   docker build -t $IMAGE_NAME:$IMAGE_TAG .
   
   # 推送镜像
   docker tag $IMAGE_NAME:$IMAGE_TAG registry.example.com/$IMAGE_NAME:$IMAGE_TAG
   docker push registry.example.com/$IMAGE_NAME:$IMAGE_TAG
   
   # 部署到Kubernetes
   kubectl apply -f k8s/namespace.yaml
   kubectl apply -f k8s/configmap.yaml
   kubectl apply -f k8s/secret.yaml
   kubectl set image deployment/short-term-memory $IMAGE_NAME=registry.example.com/$IMAGE_NAME:$IMAGE_TAG -n $NAMESPACE
   kubectl rollout status deployment/short-term-memory -n $NAMESPACE
   ```
