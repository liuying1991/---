# 文本后处理模块6A工作流

## 模块概述

文本后处理模块是信号转文字子系统的核心组件之一，负责对多模态融合后的文本进行精细化处理和优化，提高文本的可读性、准确性和语义连贯性。该模块采用先进的自然语言处理技术，包括文本纠错、语义补全、上下文优化和情感增强等功能，确保转写后的文本能够准确反映婴儿的意图和情感状态，为真实婴儿AI管家系统提供高质量的文本输入。

### 主要功能
1. 文本纠错：识别并修正文本中的语法错误、拼写错误和语音识别错误
2. 语义补全：根据上下文补全不完整的文本内容
3. 上下文优化：调整文本表达方式，使其更符合语境和逻辑
4. 情感增强：保留和增强文本中的情感色彩
5. 格式规范：统一文本格式，提高可读性
6. 个性化调整：根据婴儿特点和偏好调整文本风格

### 技术特点
1. 基于深度学习的文本纠错技术
2. 上下文感知的语义补全算法
3. 婴儿语言特点的专门优化
4. 实时处理和低延迟响应
5. 多层次文本质量评估机制

### 技术选型分析

#### 1. 文本纠错技术选型
1. **纠错方法**
   - 选项1：基于规则和词典的文本纠错
   - 选项2：基于统计机器学习的文本纠错
   - 选项3：基于深度学习的文本纠错
   - 选项4：基于预训练语言模型的文本纠错

2. **选择理由**
   - 规则和词典：简单高效，适合常见错误
   - 统计机器学习：基于数据驱动，适合特定领域
   - 深度学习：端到端学习，适合复杂错误
   - 预训练语言模型：强大的语言理解能力，适合通用纠错

3. **最终选择**
   - 采用基于预训练语言模型的文本纠错方法
   - 结合规则和词典提高常见错误处理能力
   - 针对婴儿语言特点进行微调

#### 2. 语义补全技术选型
1. **补全方法**
   - 选项1：基于N-gram的语义补全
   - 选项2：基于RNN的语义补全
   - 选项3：基于Transformer的语义补全
   - 选项4：基于预训练语言模型的语义补全

2. **选择理由**
   - N-gram：简单高效，适合局部补全
   - RNN：考虑上下文，适合序列补全
   - Transformer：强大的序列建模能力，适合长距离依赖
   - 预训练语言模型：已经学习到丰富的语言知识，适合高质量补全

3. **最终选择**
   - 采用基于预训练语言模型的语义补全方法
   - 结合上下文信息提高补全准确性
   - 针对婴儿语言特点进行优化

#### 3. 情感增强技术选型
1. **情感处理方法**
   - 选项1：基于情感词典的情感处理
   - 选项2：基于机器学习的情感处理
   - 选项3：基于深度学习的情感处理
   - 选项4：基于多模态情感分析的情感处理

2. **选择理由**
   - 情感词典：简单直观，适合基础情感分析
   - 机器学习：基于特征工程，适合特定领域
   - 深度学习：端到端学习，适合复杂情感分析
   - 多模态情感分析：结合多模态信息，提高准确性

3. **最终选择**
   - 采用基于深度学习的情感处理方法
   - 结合多模态信息提高情感分析准确性
   - 针对婴儿情感表达特点进行优化

from typing import List, Dict, Optional, Iterator
from datetime import datetime
import logging
import time

from src.models.text_models import TextProcessing

from src.ml.sentiment_models import SentimentModelFactory

logger = logging.getLogger(__name__)

class TextPostProcessingManager:
    """文本后处理管理器"""
    
    def __init__(self, config: Dict):
        """初始化文本后处理管理器"""
        self.config = config
        
        # 初始化文本处理器
        self.text_processor = TextProcessor(config.get('text_processor', {}))
        
        # 初始化文本纠错器
        self.text_corrector = TextCorrector(config.get('text_corrector', {}))
        
        # 初始化文本补全器
        self.text_completer = TextCompleter(config.get('text_completer', {}))
        
        # 初始化情感增强器
        self.sentiment_enhancer = SentimentEnhancer(config.get('sentiment_enhancer', {}))
        
        # 初始化文本格式化器
        self.text_formatter = TextFormatter(config.get('text_formatter', {}))
        
        # 初始化模型工厂
        self.correction_model_factory = CorrectionModelFactory(config.get('correction_models', {}))
        self.completion_model_factory = CompletionModelFactory(config.get('completion_models', {}))
        self.sentiment_model_factory = SentimentModelFactory(config.get('sentiment_models', {}))
        self.language_model_factory = LanguageModelFactory(config.get('language_models', {}))
        
        logger.info("TextPostProcessingManager initialized")
    
    def process_text(self, text: str, options: Dict = None) -> Dict:
        """处理文本"""
        try:
            start_time = time.time()
            
            # 解析选项
            options = options or {}
            enable_correction = options.get('enable_correction', True)
            enable_completion = options.get('enable_completion', True)
            enable_sentiment = options.get('enable_sentiment', True)
            enable_formatting = options.get('enable_formatting', True)
            language = options.get('language', 'zh')
            user_profile = options.get('user_profile', {})
            
            # 创建文本处理对象
            text_processing = TextProcessing(
                text_id=f"txt_{datetime.now().strftime('%Y%m%d_%H%M%S')}",
                original_text=text,
                processed_text=text,
                processing_steps=[],
                corrections=[],
                metadata=options,
                quality_score=0.0,
                created_at=datetime.now()
            )
            
            # 预处理文本
            processed_text = self.text_processor.preprocess(text)
            text_processing.processing_steps.append('preprocessing')
            
            # 文本纠错
            if enable_correction:
                corrected_text, corrections = self.text_corrector.correct(processed_text, language=language)
                text_processing.processed_text = corrected_text
                text_processing.corrections = corrections
                text_processing.processing_steps.append('correction')
            else:
                corrected_text = processed_text
            
            # 语义补全
            if enable_completion:
                completed_text, completions = self.text_completer.complete(corrected_text, user_profile=user_profile)
                text_processing.processed_text = completed_text
                text_processing.processing_steps.append('completion')
            else:
                completed_text = corrected_text
            
            # 情感增强
            if enable_sentiment:
                enhanced_text, enhancements = self.sentiment_enhancer.enhance(completed_text, user_profile=user_profile)
                text_processing.processed_text = enhanced_text
                text_processing.processing_steps.append('sentiment_enhancement')
            else:
                enhanced_text = completed_text
            
            # 格式规范
            if enable_formatting:
                formatted_text = self.text_formatter.format(enhanced_text, language=language)
                text_processing.processed_text = formatted_text
                text_processing.processing_steps.append('formatting')
            else:
                formatted_text = enhanced_text
            
            # 计算质量评分
            text_processing.quality_score = self._calculate_quality_score(
                text_processing.original_text,
                text_processing.processed_text,
                text_processing.corrections
            )
            text_processing.updated_at = datetime.now()
            
            # 格式化结果
            processing_time = (time.time() - start_time) * 1000
            result = {
                'text_id': text_processing.text_id,
                'original_text': text_processing.original_text,
                'processed_text': text_processing.processed_text,
                'processing_steps': text_processing.processing_steps,
                'corrections': [self._format_correction(c) for c in text_processing.corrections],
                'quality_score': text_processing.quality_score,
                'processing_time': processing_time,
                'status': 'success',
                'timestamp': datetime.now().isoformat()
            }
            
            logger.info(f"Text processing completed successfully: {text_processing.text_id}")
            return result
            
        except Exception as e:
            logger.error(f"Error processing text: {str(e)}")
            return {
                'status': 'error',
                'message': str(e),
                'timestamp': datetime.now().isoformat()
            }
    
    def correct_text(self, text: str, options: Dict = None) -> Dict:

            )

            text_processing.processed_text = corrected_text

                quality_score=0.0,
                created_at=datetime.now()
            )
            
            # 预处理文本
            processed_text = self.text_processor.preprocess(text)
            
            # 情感增强
            enhanced_text, enhancements = self.sentiment_enhancer.enhance(
                processed_text,
                user_profile=user_profile,
                enhancement_types=enhancement_types
            )
            
            # 更新文本处理对象
            text_processing.processed_text = enhanced_text
            text_processing.quality_score = self._calculate_quality_score(
                text_processing.original_text,
                text_processing.processed_text,
                text_processing.corrections
            )
            text_processing.updated_at = datetime.now()
            
            # 格式化结果
            processing_time = (time.time() - start_time) * 1000
            result = {
                'text_id': text_processing.text_id,
                'original_text': text_processing.original_text,
                'enhanced_text': text_processing.processed_text,
                'enhancements': [self._format_enhancement(e) for e in enhancements],
                'quality_score': text_processing.quality_score,
                'processing_time': processing_time,
                'status': 'success',
                'timestamp': datetime.now().isoformat()
            }
            
            logger.info(f"Sentiment enhancement completed successfully: {text_processing.text_id}")
            return result
            
        except Exception as e:
            logger.error(f"Error enhancing sentiment: {str(e)}")
            return {
                'status': 'error',
                'message': str(e),
                'timestamp': datetime.now().isoformat()
            }
    
    def batch_process(self, text_list: List[str], options: Dict = None) -> Dict:
        """批量处理文本"""
        try:
            results = []
            
            for i, text in enumerate(text_list):
                # 处理单个文本
                result = self.process_text(text, options)
                result['batch_index'] = i
                results.append(result)
            
            # 返回批量结果
            batch_result = {
                'batch_size': len(text_list),
                'results': results,
                'status': 'success',
                'timestamp': datetime.now().isoformat()
            }
            
            logger.info(f"Batch text processing completed successfully: {len(text_list)} texts")
            return batch_result
            
        except Exception as e:
            logger.error(f"Error in batch text processing: {str(e)}")
            return {
                'status': 'error',
                'message': str(e),
                'timestamp': datetime.now().isoformat()
            }
    
    def stream_process(self, text_stream: Iterator[str], options: Dict = None) -> Iterator[Dict]:
        """流式处理文本"""
        try:
            # 解析选项
            options = options or {}
            
            for text in text_stream:
                # 处理单个文本
                result = self.process_text(text, options)
                yield result
            
        except Exception as e:
            logger.error(f"Error in stream text processing: {str(e)}")
            yield {
                'status': 'error',
                'message': str(e),
                'timestamp': datetime.now().isoformat()
            }
    
    def _calculate_quality_score(self, original_text: str, processed_text: str, corrections: List) -> float:
        """计算文本处理质量评分"""
        # 简化实现，实际应该使用更复杂的算法
        base_score = 0.8
        
        # 根据修正数量调整评分
        correction_count = len(corrections)
        if correction_count == 0:
            return base_score
        elif correction_count <= 3:
            return min(base_score + 0.1, 1.0)
        else:
            return max(base_score - 0.1, 0.5)
    
    def _format_correction(self, correction) -> Dict:
        """格式化修正记录"""
        return {
            'correction_id': correction.correction_id,
            'original_segment': correction.original_segment,
            'corrected_segment': correction.corrected_segment,
            'correction_type': correction.correction_type,
            'confidence': correction.confidence
        }
    
    def _format_completion(self, completion) -> Dict:
        """格式化补全记录"""
        return {
            'completion_id': completion.completion_id,
            'original_segment': completion.original_segment,
            'completed_segment': completion.completed_segment,
            'completion_type': completion.completion_type,
            'confidence': completion.confidence
        }
    
    def _format_enhancement(self, enhancement) -> Dict:
        """格式化增强记录"""
        return {
            'enhancement_id': enhancement.enhancement_id,
            'original_segment': enhancement.original_segment,
            'enhanced_segment': enhancement.enhanced_segment,
            'sentiment_type': enhancement.sentiment_type,
            'sentiment_strength': enhancement.sentiment_strength,
            'confidence': enhancement.confidence
        }
