## 2. 安全架构概述

### 2.1 安全设计原则

#### 2.1.1 核心安全原则
1. **最小权限原则**：每个组件只授予完成任务所需的最小权限
2. **纵深防御原则**：在系统多个层面部署安全控制措施
3. **零信任原则**：不默认信任任何内部或外部请求，所有请求都需要验证
4. **安全默认原则**：系统默认配置应为最安全状态
5. **完整性与机密性原则**：确保数据在传输和存储过程中的完整性和机密性

#### 2.1.2 AI系统特有安全原则
1. **模型安全原则**：确保AI模型免受对抗性攻击和模型窃取
2. **数据隐私原则**：保护用户数据隐私，防止敏感信息泄露
3. **决策透明原则**：确保AI决策过程可解释、可审计
4. **公平性原则**：防止AI系统产生歧视性或偏见性结果
5. **可追溯性原则**：确保AI系统的所有行为可被追踪和审计

### 2.2 安全架构层次

#### 2.2.1 基础设施安全层
- 物理安全：数据中心物理访问控制
- 网络安全：网络隔离、防火墙、入侵检测
- 主机安全：操作系统加固、主机入侵检测
- 容器安全：容器镜像扫描、运行时保护

#### 2.2.2 应用安全层
- 身份认证与授权：多因素认证、基于角色的访问控制
- API安全：API网关、速率限制、输入验证
- 数据安全：数据加密、数据脱敏、数据备份
- 会话安全：安全会话管理、会话超时控制

#### 2.2.3 AI模型安全层
- 模型保护：模型加密、访问控制
- 输入验证：对抗性样本检测、异常输入过滤
- 输出控制：敏感信息过滤、输出内容审核
- 模型完整性：模型签名验证、完整性校验

#### 2.3.2 加密服务组件
```python
# 加密服务组件设计
from cryptography.hazmat.primitives.ciphers import Cipher, algorithms, modes
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC
from cryptography.hazmat.backends import default_backend
import os
import base64

class EncryptionService:
    def __init__(self):
        self.key_manager = KeyManager()
    
    def encrypt_data(self, data, key_id=None):
        """数据加密"""
        key = self.key_manager.get_key(key_id)
        iv = os.urandom(16)
        cipher = Cipher(
            algorithms.AES(key),
            modes.CBC(iv),
            backend=default_backend()
        )
        encryptor = cipher.encryptor()
        
        # 填充数据
        padded_data = self._pad_data(data.encode())
        encrypted_data = encryptor.update(padded_data) + encryptor.finalize()
        
        # 返回IV + 加密数据
        return base64.b64encode(iv + encrypted_data).decode()
    
    def decrypt_data(self, encrypted_data, key_id=None):
        """数据解密"""
        key = self.key_manager.get_key(key_id)
        encrypted_data = base64.b64decode(encrypted_data.encode())
        
        # 提取IV和加密数据
        iv = encrypted_data[:16]
        ciphertext = encrypted_data[16:]
        
        cipher = Cipher(
            algorithms.AES(key),
            modes.CBC(iv),
            backend=default_backend()
        )
        decryptor = cipher.decryptor()
        padded_data = decryptor.update(ciphertext) + decryptor.finalize()
        
        # 去除填充
        data = self._unpad_data(padded_data).decode()
        return data
    
    def _pad_data(self, data):
        """数据填充"""
        block_size = 16
        padding_length = block_size - (len(data) % block_size)
        padding = bytes([padding_length] * padding_length)
        return data + padding
    
    def _unpad_data(self, padded_data):
        """去除数据填充"""
        padding_length = padded_data[-1]
        return padded_data[:-padding_length]

class KeyManager:
    def __init__(self):
        self.keys = {}
        self.master_key = self._generate_master_key()
    
    def _generate_master_key(self):
        """生成主密钥"""
        return os.urandom(32)  # 256位密钥
    
    def get_key(self, key_id=None):
        """获取密钥"""
        if key_id is None:
            key_id = 'default'
        
        if key_id not in self.keys:
            self.keys[key_id] = self._derive_key(key_id)
        
        return self.keys[key_id]
    
    def _derive_key(self, key_id):
        """派生密钥"""
        kdf = PBKDF2HMAC(
            algorithm=hashes.SHA256(),
            length=32,
            salt=key_id.encode(),
            iterations=100000,
            backend=default_backend()
        )
        return kdf.derive(self.master_key)
```

## 3. 威胁分析与风险评估

### 3.1 威胁模型

#### 3.1.1 威胁分类
1. **外部威胁**
   - 恶意攻击者：试图未经授权访问系统或数据
   - 自动化攻击：使用工具自动扫描和攻击系统漏洞
   - 社会工程：通过欺骗手段获取敏感信息或访问权限

2. **内部威胁**
   - 恶意内部人员：故意破坏系统或窃取数据
   - 无意内部人员：因疏忽或错误导致安全问题
   - 权限滥用：滥用合法权限执行未授权操作

3. **系统威胁**
   - 软件漏洞：系统或应用程序中的安全缺陷
   - 配置错误：不安全的系统配置
   - 硬件故障：导致数据丢失或服务中断

4. **AI特有威胁**
   - 对抗性攻击：通过特殊输入误导AI模型
   - 模型窃取：窃取训练好的AI模型
   - 数据投毒：污染训练数据影响模型性能

#### 3.1.2 攻击向量
1. **网络攻击向量**
   - DDoS攻击：通过大量请求使系统不可用
   - 中间人攻击：拦截和篡改通信数据
   - 网络嗅探：捕获网络传输的敏感数据

2. **应用攻击向量**
   - SQL注入：通过恶意SQL查询获取或修改数据
   - XSS攻击：在网页中注入恶意脚本
   - CSRF攻击：诱使用户执行未授权操作

3. **AI攻击向量**
   - 对抗性样本：精心构造的输入导致模型错误分类
   - 模型逆向：通过分析模型输出推断模型参数
   - 成员推理：判断特定数据是否用于训练模型

### 3.2 风险评估

#### 3.2.1 风险评估方法
采用DREAD风险评估模型，从以下维度评估风险：
- **Damage Potential (损害潜力)**：攻击成功可能造成的损害程度
- **Reproducibility (可重现性)**：攻击步骤的可重现程度
- **Exploitability (可利用性)**：发起攻击的难易程度
- **Affected Users (受影响用户)**：受影响用户的比例
- **Discoverability (可发现性)**：漏洞被发现的难易程度

#### 3.2.2 风险评估矩阵
| 威胁类型 | 损害潜力 | 可重现性 | 可利用性 | 受影响用户 | 可发现性 | 风险等级 |
|----------|----------|----------|----------|------------|----------|----------|
| SQL注入 | 高 | 高 | 高 | 中 | 中 | 高 |
| XSS攻击 | 中 | 高 | 高 | 高 | 高 | 高 |
| DDoS攻击 | 中 | 高 | 高 | 高 | 高 | 高 |
| 对抗性攻击 | 高 | 中 | 中 | 中 | 低 | 中 |
| 数据泄露 | 高 | 中 | 中 | 高 | 低 | 中 |
| 模型窃取 | 高 | 低 | 低 | 低 | 低 | 低 |

#### 3.2.3 风险处理策略
1. **高风险**：立即采取缓解措施，优先处理
2. **中风险**：制定处理计划，定期评估
3. **低风险**：接受风险，监控变化

```python
# SSO服务实现
import jwt
import uuid
from datetime import datetime, timedelta
from urllib.parse import urlencode, parse_qs

class SSOService:
    def __init__(self):
        self.auth_service = AuthenticationService()
        self.client_store = ClientStore()
        self.token_store = TokenStore()
        self.jwt_secret = "your-jwt-secret-key"
        self.jwt_algorithm = "HS256"
    
    def authorize(self, client_id, redirect_uri, response_type="code", scope=None, state=None):
        """授权请求"""
        # 验证客户端
        client = self.client_store.get_client(client_id)
        if not client:
            return {"error": "invalid_client"}
        
        # 验证重定向URI
        if redirect_uri not in client.redirect_uris:
            return {"error": "invalid_redirect_uri"}
        
        # 生成授权请求
        auth_request = {
            "request_id": str(uuid.uuid4()),
            "client_id": client_id,
            "redirect_uri": redirect_uri,
            "response_type": response_type,
            "scope": scope or "openid profile",
            "state": state
        }
        
        # 存储授权请求
        self._store_auth_request(auth_request)
        
        # 返回登录页面
        return {
            "action": "redirect_to_login",
            "auth_request_id": auth_request["request_id"]
        }
    
    def process_login(self, auth_request_id, username, password, mfa_code=None):
        """处理登录"""
        # 获取授权请求
        auth_request = self._get_auth_request(auth_request_id)
        if not auth_request:
            return {"error": "invalid_request"}
        
        # 验证用户凭据
        auth_result = self.auth_service.authenticate(username, password, mfa_code)
        if not auth_result["success"]:
            return auth_result
        
        # 标记用户已认证
        auth_request["user_id"] = auth_result["user_id"]
        auth_request["authenticated"] = True
        self._update_auth_request(auth_request)
        
        # 询问用户是否授权
        return {
            "action": "ask_consent",
            "client_id": auth_request["client_id"],
            "scope": auth_request["scope"]
        }
    
    def grant_consent(self, auth_request_id, granted):
        """处理用户授权"""
        # 获取授权请求
        auth_request = self._get_auth_request(auth_request_id)
        if not auth_request or not auth_request.get("authenticated"):
            return {"error": "invalid_request"}
        
        client = self.client_store.get_client(auth_request["client_id"])
        
        if granted:
            if auth_request["response_type"] == "code":
                # 授权码流程
                auth_code = self._generate_auth_code(
                    auth_request["user_id"],
                    auth_request["client_id"],
                    auth_request["redirect_uri"],
                    auth_request["scope"]
                )
                
                # 构建重定向URL
                params = {
                    "code": auth_code,
                    "state": auth_request.get("state")
                }
                redirect_url = f"{auth_request['redirect_uri']}?{urlencode(params)}"
                
                return {"action": "redirect", "url": redirect_url}
            else:
                # 隐式流程
                access_token = self._generate_access_token(
                    auth_request["user_id"],
                    auth_request["client_id"],
                    auth_request["scope"]
                )
                
                # 构建重定向URL
                params = {
                    "access_token": access_token["token"],
                    "token_type": access_token["token_type"],
                    "expires_in": access_token["expires_in"],
                    "state": auth_request.get("state")
                }
                fragment = f"#{urlencode(params)}"
                redirect_url = f"{auth_request['redirect_uri']}{fragment}"
                
                return {"action": "redirect", "url": redirect_url}
        else:
            # 用户拒绝授权
            params = {
                "error": "access_denied",
                "state": auth_request.get("state")
            }
            redirect_url = f"{auth_request['redirect_uri']}?{urlencode(params)}"
            
            return {"action": "redirect", "url": redirect_url}
    
    def exchange_code_for_token(self, client_id, client_secret, code, redirect_uri):
        """用授权码换取访问令牌"""
        # 验证客户端
        client = self.client_store.get_client(client_id)
        if not client or client.client_secret != client_secret:
            return {"error": "invalid_client"}
        
        # 验证授权码
        auth_code_data = self._get_auth_code(code)
        if not auth_code_data or auth_code_data["client_id"] != client_id or auth_code_data["redirect_uri"] != redirect_uri:
            return {"error": "invalid_grant"}
        
        # 生成访问令牌
        access_token = self._generate_access_token(
            auth_code_data["user_id"],
            client_id,
            auth_code_data["scope"]
        )
        
        # 删除授权码
        self._delete_auth_code(code)
        
        return access_token
    
    def _generate_auth_code(self, user_id, client_id, redirect_uri, scope):
```

```python
        self.policy_id = policy_id
        self.name = name
        self.description = description
        self.rules = []

        """评估策略"""
        # 所有规则都必须满足
        return all(rule.evaluate(attributes) for rule in self.rules)

class Rule:
    def __init__(self, target: str, condition: str):
        self.target = target
        self.condition = condition
    
    def evaluate(self, attributes: Dict[str, Dict[str, Any]]) -> bool:
        """评估规则"""
        try:
            # 解析目标属性
            attr_type, attr_name = self.target.split(".", 1)
            
            if attr_type not in attributes or attr_name not in attributes[attr_type]:
                return False
            
            # 获取属性值
            attr_value = attributes[attr_type][attr_name]
            
            # 评估条件
            return eval(self.condition, {"value": attr_value})
        except:
            return False
```

```python
# 存储加密实现
import os
import sqlite3
from cryptography.hazmat.primitives.ciphers import Cipher, algorithms, modes
from cryptography.hazmat.primitives import hashes
from cryptography.hazmat.primitives.kdf.pbkdf2 import PBKDF2HMAC
from cryptography.hazmat.backends import default_backend
import base64

class EncryptedDatabase:
    def __init__(self, db_path, password):
        self.db_path = db_path
        self.key = self._derive_key(password)
        self.connection = None
    
    def _derive_key(self, password):
        """从密码派生密钥"""
        salt = b'infant_ai_database_salt'  # 在实际应用中应该使用随机盐
        kdf = PBKDF2HMAC(
            algorithm=hashes.SHA256(),
            length=32,
            salt=salt,
            iterations=100000,
            backend=default_backend()
        )
        return kdf.derive(password.encode())
    
    def _encrypt_data(self, data):
        """加密数据"""
        iv = os.urandom(16)
        cipher = Cipher(
            algorithms.AES(self.key),
            modes.CBC(iv),
            backend=default_backend()
        )
        encryptor = cipher.encryptor()
        
        # 填充数据
        padded_data = self._pad_data(data.encode())
        encrypted_data = encryptor.update(padded_data) + encryptor.finalize()
        
        # 返回IV + 加密数据
        return base64.b64encode(iv + encrypted_data).decode()
    
    def _decrypt_data(self, encrypted_data):
        """解密数据"""
        encrypted_data = base64.b64decode(encrypted_data.encode())
        
        # 提取IV和加密数据
        iv = encrypted_data[:16]
        ciphertext = encrypted_data[16:]
        
        cipher = Cipher(
            algorithms.AES(self.key),
            modes.CBC(iv),
            backend=default_backend()
        )
        decryptor = cipher.decryptor()
        padded_data = decryptor.update(ciphertext) + decryptor.finalize()
        
        # 去除填充
        data = self._unpad_data(padded_data).decode()
        return data
    
    def _pad_data(self, data):
        """数据填充"""
        block_size = 16
        padding_length = block_size - (len(data) % block_size)
        padding = bytes([padding_length] * padding_length)
        return data + padding
    
    def _unpad_data(self, padded_data):
        """去除数据填充"""
        padding_length = padded_data[-1]
        return padded_data[:-padding_length]
    
    def connect(self):
        """连接到加密数据库"""
        self.connection = sqlite3.connect(self.db_path)
        self.connection.row_factory = sqlite3.Row
        return self.connection
    
    def execute(self, query, params=None):
        """执行SQL查询"""
        if not self.connection:
            self.connect()
        
        cursor = self.connection.cursor()
        if params:
            cursor.execute(query, params)
        else:
            cursor.execute(query)
        
        return cursor
    
    def insert_encrypted(self, table, data):
        """插入加密数据"""
        columns = list(data.keys())
        encrypted_data = {col: self._encrypt_data(str(data[col])) for col in columns}
        
        placeholders = ', '.join(['?'] * len(columns))
        columns_str = ', '.join(columns)
        values = [encrypted_data[col] for col in columns]
        
        query = f"INSERT INTO {table} ({columns_str}) VALUES ({placeholders})"
        self.execute(query, values)
        self.connection.commit()
    
    def select_decrypted(self, table, columns=None, where=None, params=None):
        """查询并解密数据"""
        if columns is None:
            columns = ['*']
        
        columns_str = ', '.join(columns)
        query = f"SELECT {columns_str} FROM {table}"
        
        if where:
            query += f" WHERE {where}"
        
        cursor = self.execute(query, params)
        rows = cursor.fetchall()
        
        # 解密数据
        result = []
        for row in rows:
            decrypted_row = {}
            for col in row.keys():
                if col in columns or '*' in columns:
                    try:
                        decrypted_row[col] = self._decrypt_data(row[col])
                    except:
                        decrypted_row[col] = row[col]
                else:
                    decrypted_row[col] = row[col]
            result.append(decrypted_row)
        
        return result
    
    def close(self):
        """关闭数据库连接"""
        if self.connection:
            self.connection.close()
            self.connection = None

class FileEncryption:
    def __init__(self, key):
        self.key = key
    
    def encrypt_file(self, input_file, output_file):
        """加密文件"""
        iv = os.urandom(16)
        cipher = Cipher(
            algorithms.AES(self.key),
            modes.CBC(iv),
            backend=default_backend()
        )
        encryptor = cipher.encryptor()
        
        with open(input_file, 'rb') as infile, open(output_file, 'wb') as outfile:
            # 写入IV
            outfile.write(iv)
            
            # 加密文件内容
            while True:
                chunk = infile.read(16 * 1024)  # 16KB chunks
                if not chunk:
                    break
                
                # 填充最后一个块
                if len(chunk) % 16 != 0:
                    padding_length = 16 - (len(chunk) % 16)
                    chunk += bytes([padding_length] * padding_length)
                
                encrypted_chunk = encryptor.update(chunk)
                outfile.write(encrypted_chunk)
            
            # 完成加密
            outfile.write(encryptor.finalize())
    
    def decrypt_file(self, input_file, output_file):
        """解密文件"""
        with open(input_file, 'rb') as infile, open(output_file, 'wb') as outfile:
            # 读取IV
            iv = infile.read(16)
            
            cipher = Cipher(
                algorithms.AES(self.key),
                modes.CBC(iv),
                backend=default_backend()
            )
            decryptor = cipher.decryptor()
            
            # 解密文件内容
            while True:
                chunk = infile.read(16 * 1024 + 16)  # 可能包含填充的块
                if not chunk:
                    break
                
                decrypted_chunk = decryptor.update(chunk)
```

```python
# 差分隐私实现
import numpy as np
import random
from typing import List, Dict, Any, Callable

class DifferentialPrivacy:
    def __init__(self, epsilon: float = 1.0):
        self.epsilon = epsilon
    
    def laplace_mechanism(self, true_value: float, sensitivity: float) -> float:
        """拉普拉斯机制"""
        if sensitivity == 0:
            return true_value
        
        scale = sensitivity / self.epsilon
        noise = np.random.laplace(0, scale)
        return true_value + noise
    
    def exponential_mechanism(self, options: List[Any], score_func: Callable, sensitivity: float) -> Any:
        """指数机制"""
        scores = [score_func(option) for option in options]
        
        # 计算概率
        max_score = max(scores)
        probabilities = [np.exp(self.epsilon * (score - max_score) / (2 * sensitivity)) for score in scores]
        
        # 归一化概率
        total = sum(probabilities)
        probabilities = [p / total for p in probabilities]
        
        # 根据概率选择
        return np.random.choice(options, p=probabilities)
    
    def count_query(self, data: List[Any], condition: Callable) -> int:
        """计数查询"""
        count = sum(1 for item in data if condition(item))
        return int(self.laplace_mechanism(count, sensitivity=1))
    
    def sum_query(self, data: List[float], condition: Callable = None) -> float:
        """求和查询"""
        if condition:
            filtered_data = [item for item in data if condition(item)]
        else:
            filtered_data = data
        
        true_sum = sum(filtered_data)
        
        # 计算敏感度（假设每个值在[0, 1]范围内）
        sensitivity = len(filtered_data) * 1.0
        return self.laplace_mechanism(true_sum, sensitivity)
    
    def average_query(self, data: List[float], condition: Callable = None) -> float:
        """平均值查询"""
        if condition:
            filtered_data = [item for item in data if condition(item)]
        else:
            filtered_data = data
        
        if not filtered_data:
            return 0.0
        
        true_avg = sum(filtered_data) / len(filtered_data)
        
        # 计算敏感度（假设每个值在[0, 1]范围内）
        sensitivity = 1.0 / len(filtered_data)
        return self.laplace_mechanism(true_avg, sensitivity)
    
    def histogram_query(self, data: List[Any], bin_func: Callable) -> Dict[Any, int]:
        """直方图查询"""
        bins = {}
        for item in data:
            bin_key = bin_func(item)
            if bin_key not in bins:
                bins[bin_key] = 0
            bins[bin_key] += 1
        
        # 为每个bin添加噪声
        noisy_bins = {}
        for bin_key, count in bins.items():
            noisy_bins[bin_key] = int(self.laplace_mechanism(count, sensitivity=1))
        
        return noisy_bins

class PrivateDataRelease:
```

```python
# 联邦学习实现
import numpy as np
import torch
import torch.nn as nn
import torch.optim as optim
from typing import List, Dict, Any, Tuple
import copy

class FederatedLearning:
    def __init__(self, global_model: nn.Module, device='cpu'):
        self.global_model = global_model.to(device)
        self.device = device
        self.client_models = {}
    
    def create_client_model(self, client_id: str) -> nn.Module:
        """为客户端创建模型副本"""
        client_model = copy.deepcopy(self.global_model)
        self.client_models[client_id] = client_model.to(self.device)
        return client_model
    
    def client_train(self, client_id: str, train_loader, epochs=5, lr=0.01):
        """客户端训练"""
        if client_id not in self.client_models:
            self.create_client_model(client_id)
        
        model = self.client_models[client_id]
        model.train()
        
        criterion = nn.CrossEntropyLoss()
        optimizer = optim.SGD(model.parameters(), lr=lr)
        
        for epoch in range(epochs):
            for inputs, labels in train_loader:
                inputs, labels = inputs.to(self.device), labels.to(self.device)
                
                optimizer.zero_grad()
                outputs = model(inputs)
                loss = criterion(outputs, labels)
                loss.backward()
                optimizer.step()
        
        return model
    
    def federated_averaging(self, client_ids: List[str], client_weights: List[float] = None):
        """联邦平均聚合"""
        if client_weights is None:
            client_weights = [1.0 / len(client_ids)] * len(client_ids)
        
        # 初始化全局模型参数
        global_params = {}
        for name, param in self.global_model.named_parameters():
            global_params[name] = torch.zeros_like(param.data)
        
        # 加权平均客户端模型参数
        for client_id, weight in zip(client_ids, client_weights):
            if client_id not in self.client_models:
                continue
                
            client_model = self.client_models[client_id]
            for name, param in client_model.named_parameters():
                global_params[name] += weight * param.data
        
        # 更新全局模型
        for name, param in self.global_model.named_parameters():
            param.data = global_params[name]
        
        return self.global_model
    
    def distribute_global_model(self, client_ids: List[str]):
        """分发全局模型到客户端"""
        for client_id in client_ids:
            if client_id not in self.client_models:
                self.create_client_model(client_id)
            
            # 复制全局模型参数到客户端模型
            client_model = self.client_models[client_id]
            for name, param in self.global_model.named_parameters():
                client_model.state_dict()[name].copy_(param.data)
    
    def evaluate_global_model(self, test_loader):
        """评估全局模型"""
        self.global_model.eval()
        correct = 0
        total = 0
        
        with torch.no_grad():
            for inputs, labels in test_loader:
                inputs, labels = inputs.to(self.device), labels.to(self.device)
                outputs = self.global_model(inputs)
                _, predicted = torch.max(outputs.data, 1)
                total += labels.size(0)
                correct += (predicted == labels).sum().item()
        
        accuracy = 100 * correct / total
        return accuracy

# 差分隐私联邦学习
class DPFederatedLearning(FederatedLearning):
    def __init__(self, global_model: nn.Module, epsilon: float = 1.0, delta: float = 1e-5, clip_norm=1.0, device='cpu'):
        super().__init__(global_model, device)
        self.epsilon = epsilon
        self.delta = delta
        self.clip_norm = clip_norm
    
    def _clip_model_updates(self, model: nn.Module, reference_model: nn.Module):
        """裁剪模型更新"""
        total_norm = 0.0
        
        # 计算模型更新的L2范数
        for name, param in model.named_parameters():
            if name in reference_model.state_dict():
                diff = param.data - reference_model.state_dict()[name]
                total_norm += torch.sum(diff ** 2).item()
        
        total_norm = np.sqrt(total_norm)
        
        # 裁剪更新
        if total_norm > self.clip_norm:
            clip_factor = self.clip_norm / total_norm
            for name, param in model.named_parameters():
                if name in reference_model.state_dict():
                    diff = param.data - reference_model.state_dict()[name]
                    param.data = reference_model.state_dict()[name] + clip_factor * diff
        
        return total_norm
    
    def _add_noise_to_model(self, model: nn.Module, sensitivity: float):
        """向模型添加噪声"""
        for name, param in model.named_parameters():
            # 计算噪声规模
            noise_scale = sensitivity * np.sqrt(2 * np.log(1.25 / self.delta)) / self.epsilon
            
            # 添加高斯噪声
            noise = torch.normal(0, noise_scale, size=param.data.shape).to(self.device)
            param.data += noise
    
    def dp_federated_averaging(self, client_ids: List[str], client_weights: List[float] = None):
        """差分隐私联邦平均聚合"""
        if client_weights is None:
            client_weights = [1.0 / len(client_ids)] * len(client_ids)
        
        # 计算敏感度
        sensitivity = 2.0 * self.clip_norm  # 模型参数的敏感度
        
        # 初始化全局模型参数
        global_params = {}
        for name, param in self.global_model.named_parameters():
            global_params[name] = torch.zeros_like(param.data)
        
        # 加权平均客户端模型参数
        for client_id, weight in zip(client_ids, client_weights):
            if client_id not in self.client_models:
                continue
            
            client_model = self.client_models[client_id]
            
            # 裁剪模型更新
            self._clip_model_updates(client_model, self.global_model)
            
            # 添加噪声
            self._add_noise_to_model(client_model, sensitivity)
            
            # 累积加权更新
            for name, param in client_model.named_parameters():
                global_params[name] += weight * param.data
        
        # 更新全局模型
        for name, param in self.global_model.named_parameters():
            param.data = global_params[name]
        
        return self.global_model

class FederatedLearningServer:
    def __init__(self, global_model: nn.Module, use_dp=False, epsilon=1.0):
        if use_dp:
            self.fl = DPFederatedLearning(global_model, epsilon=epsilon)
        else:
            self.fl = FederatedLearning(global_model)
        
        self.rounds = 0
        self.history = []
    
    def federated_round(self, client_data: Dict[str, Tuple], epochs=5, lr=0.01):
        """执行一轮联邦学习"""
        client_ids = list(client_data.keys())
```

```python
        if self._should_notify_data_subjects(breach_record):
            self._notify_data_subjects(breach_id)
            breach_record["notification_to_data_subjects"] = True
            if not breach_record["notification_date"]:
                breach_record["notification_date"] = datetime.utcnow().isoformat()
        
        return breach_id
    
    def _should_notify_authorities(self, breach_record: Dict[str, Any]) -> bool:
        """判断是否需要通知当局"""
        # 如果泄露可能导致高风险，需要通知当局
        high_risk_factors = [
            len(breach_record["affected_users"]) > 100,  # 影响大量用户
            "special_category_data" in breach_record["affected_data"],  # 特殊类别数据
            "financial_data" in breach_record["affected_data"]  # 财务数据
        ]
        
        return any(high_risk_factors)
    
    def _should_notify_data_subjects(self, breach_record: Dict[str, Any]) -> bool:
        """判断是否需要通知数据主体"""
        # 如果泄露可能导致高风险，需要通知数据主体
        high_risk_factors = [
            "identity_theft_risk" in breach_record.get("risks", []),  # 身份盗用风险
            "financial_loss_risk" in breach_record.get("risks", []),  # 财务损失风险
            "discrimination_risk" in breach_record.get("risks", [])  # 歧视风险
        ]
        
        return any(high_risk_factors)
    
    def _notify_authorities(self, breach_id: str):
        """通知当局"""
        # 在实际应用中，这里应该通知相关数据保护当局
        breach_record = self.breach_notifications[breach_id]
        print(f"通知当局关于数据泄露 {breach_id}: {breach_record['description']}")
    
    def _notify_data_subjects(self, breach_id: str):
        """通知数据主体"""
        # 在实际应用中，这里应该通知受影响的数据主体
        breach_record = self.breach_notifications[breach_id]
        print(f"通知数据主体关于数据泄露 {breach_id}: {breach_record['description']}")
    
    def get_consent_records(self, user_id: str = None) -> List[Dict[str, Any]]:
        """获取同意记录"""
        if user_id:
            return [record for record in self.consent_records.values() if record["user_id"] == user_id]
        return list(self.consent_records.values())
    
    def get_data_subject_requests(self, user_id: str = None) -> List[Dict[str, Any]]:
        """获取数据主体请求"""
        if user_id:
            return [request for request in self.data_subject_requests.values() if request["user_id"] == user_id]
        return list(self.data_subject_requests.values())
    
    def get_data_processing_records(self, controller: str = None) -> List[Dict[str, Any]]:
        """获取数据处理记录"""
        if controller:
            return [record for record in self.data_processing_records.values() if record["controller"] == controller]
        return list(self.data_processing_records.values())
    
    def get_breach_notifications(self, status: str = None) -> List[Dict[str, Any]]:
        """获取数据泄露通知"""
        if status:
            return [breach for breach in self.breach_notifications.values() 
                   if breach.get("status", "open") == status]
        return list(self.breach_notifications.values())
```
